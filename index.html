jQuery geoRegionalización

Github Repo

Este sencillo plugin permite generar dependencia entre 2 <select> y entre ellos mostrar la relación region / comuna que conforma la distribución geopolítica de la República de Chile.

demo base:

			
$('#region').geoRegionalizacion({
   regionDependiente: '#comuna'
});

demo avanzado:

			
$('#region-avanzado').geoRegionalizacion({
   regionDependiente: '#comuna-avanzado',
      onRegionSelect: function(){
      console.log($(this).val());
   },
   onComunaSelect: function() {
      console.log($(this).val());
   },
   onCreate: function(){
      $('#region-avanzado, #comuna-avanzado').selectric('refresh');
   }
});
